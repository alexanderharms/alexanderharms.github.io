Text labeling

Cursedlabeling

When working with machine learning it's necessary to have a labeled dataset to train your model on. If you're out of luck, then you need to label this dataset yourself. With 'cursedlabeling' I have written a small tool in Python using the 'curses' library to make the labeling of texts a little easier.

Check out the repository for cursedlabeling
